Q> 다음 배열 arr 에서 숫자 1, 2, 3을 슬라이싱 하는 것으로 알맞은 코드를 고르세요.

```python
arr = np.arange(7)
```

```shell
[0, 1, 2, 3, 4, 5, 6, 7]
```

1. ```arr[1:3]```
2. ```arr[1:4]```
3. ```arr[:4]```
4. ```arr[:3]```

A> 2

해설>
<br>슬라이싱의 시작 인덱스는 그대로지만, 끝 인덱스는 슬라이싱 하고자 하는 요소의 끝 자리에 +1을 해줘야합니다.<br>
따라서 ```arr[1:4]```가 정답입니다.
