Q> 

다음 코드의 실행 결과로 알맞은 것을 고르세요.

```python
A = np.arange(6).reshape(3,2)
B = np.arange(3).reshape(3,1)
print(A+B)
```

```python

 [3 4]
 [6 7]]
 ```

2번 :

```python
[[1 2]
 [4 5]
 [7 8]]
 ```

```python
Error
```

1

해설>

배열 A의 모양은 다음과 같습니다.

```python
[[0 1]
 [2 3]
 [4 5]]
 ```

배열 B의 모양은 다음과 같습니다.

```python
[[0]
 [1]
 [2]]
 ```

```python
[[0 1]
 [3 4]
 [6 7]]
 ```
